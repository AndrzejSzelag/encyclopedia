## Requirements to run the project

* Install python and pip.

* After pip was installed, run:
    
    ````
    pip install django
    ````
    ````
    pip install markdown2
    ````
    ````
    pip install django-markdownify
    ````
* In the root folder, execute: 
    ````
    python manage.py runserver
    ````
